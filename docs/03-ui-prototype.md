### 1.2 设计规范

| 项目 | 规范 |
|------|------|
| 组件库 | **shadcn/ui** + **Tailwind CSS 4.x** + **lucide-react** 图标 + **Recharts** 图表 |
| 主色调 | 浅色模式主色 `hsl(222, 47%, 51%)` 蓝色系，深色模式主色 `hsl(28, 72%, 56%)` 暖橙色系；木色 `hsl(36, 70%, 31%)` 仅作辅助装饰色 |
| 背景色 | 浅色模式 `hsl(0, 0%, 98%)`，深色模式 `hsl(222, 47%, 11%)` |
| 侧边栏宽度 | 折叠 64px / 展开 240px |
| 字体 | 主字体 **Inter**，中文回退 **Noto Sans SC**，越南语回退 **Noto Sans Vietnamese**；开发时使用 Web Font 加载，不依赖系统字体栈 |
| 圆角 | 卡片 `--radius` (0.5rem)、按钮 `--radius`、输入框 `calc(--radius - 2px)` |
| 间距规范 | 页面内边距 24px，卡片间距 16px，表单行间距 20px |
| 表格行高 | 48px |
| 动画 | 页面切换 200ms ease，弹窗 150ms scale-up，微交互 transition-all |
| 国际化 | 顶栏语言切换器（🇨🇳中文 / 🇻🇳 Tiếng Việt / 🇺🇸 English） |

### 1.3 响应式策略

桌面端应用，主要适配两种显示模式：

| 模式 | 窗口尺寸 | 侧边栏 | 布局 |
|------|---------|--------|------|
| 标准 | ≥ 1280px | 展开 | 正常布局 |
| 紧凑 | < 1280px | 折叠为图标 | 内容占满宽度 |

> **最小窗口尺寸**：`minWidth: 1024px, minHeight: 768px`。低于此尺寸时 Tauri 窗口不允许继续缩小。

**交互说明**：
- 标 `*` 字段为必填项，提交时校验并高亮错误字段

- 分类选择为级联选择器（Cascader），展示树形结构
- 图片上传后本地预览，存储在应用数据目录
- 保存成功后自动关闭抽屉，列表刷新并定位到新记录
- 编辑时预填当前值，字段不可编辑的灰显（如编码）
- 辅助单位为可选项，选择后"换算比例"变为必填
- 批次追踪模式默认为"不追踪"，设为"必填"后该物料入库时必须录入批次号
- 已有库存的物料不可将追踪模式从"必填"降级为"不追踪"

> **抽屉（Drawer）与弹窗（Modal）选择标准**：

> - 字段数 > 6 个或含复杂布局的表单 → Drawer（如物料新增/编辑、采购单新建）
> - 信息展示（非编辑）→ Modal（如详情预览、确认对话框）
